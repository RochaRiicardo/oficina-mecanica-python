#  Sistema Oficina Mecânica

Sistema de gerenciamento de oficina mecânica desenvolvido em Python utilizando SQLite como banco de dados.

## Funcionalidades

- Cadastro de clientes
- Cadastro de veículos
- Ordens de serviço
- Consultas
- Relatórios
- Banco de dados SQLite

## Tecnologias

- Python 3
- SQLite

## Estrutura

Sistema OficinaMecanica/

── main.py
── banco.py
── cliente.py
── veiculo.py
── ordem_servico.py
── consultas.py
── relatorios.py
── constantes.py
── menu.py

## Como executar

```bash
python main.py
```

## Objetivo

Este projeto foi desenvolvido com o objetivo de praticar programação em Python, organização modular do código e manipulação de banco de dados SQLite.
